# MedSignal demo graphics

These are 1920 × 1080 SVG cards for the non-product-screen moments in the demo.
They use only synthetic clinical context and are safe to show alongside the live
application recordings.

## Suggested order

1. `01-problem-scale.svg` — open the video with this card.
2. `02-maria-context.svg` — introduce María, then cut to the live patient timeline.
3. `03-local-privacy.svg` — use before the live workflow recording.
4. `04-agentic-workflow.svg` — use while the agent trace/tool calls run.
5. `05-contradiction.svg` — use immediately before or after the live contradiction screen.
6. `06-prescription-risk.svg` — use with the prescription/OCR recording.
7. `07-audit-trail.svg` — use with the correction flow recording.
8. `08-patient-language.svg` — use before the patient-facing Spanish view.
9. `09-safety-boundaries.svg` — use during the GPT-5.6/Codex explanation.
10. `10-closing.svg` — final card.

Every card is editable in Figma, Keynote, Canva, Illustrator, or a browser.
For a clean cut, leave each card onscreen for 2–4 seconds, with the exception of
`04-agentic-workflow.svg`, which can remain visible behind the active trace for
about 6 seconds.

The source figures on the first card are from WHO Patient Safety and WHO
Medication Without Harm. Keep the small source line in the exported video.
